## Source 2 – Express Entry: Language test results (program minima)
URL: https://www.canada.ca/en/immigration-refugees-citizenship/services/immigrate-canada/express-entry/documents/language-test.html  
DATE_ACCESSED: 2025-12-07  
TYPE: HTML page  
NOTES:
- Lists accepted tests (CELPIP General, IELTS General Training, PTE Core, TEF Canada, TCF Canada).
- Provides minimum CLB/NCLC levels for programs:
  - CEC: TEER 0/1 → CLB/NCLC 7; TEER 2/3 → CLB/NCLC 5.
  - FSW: first official language CLB/NCLC 7; second official language CLB/NCLC 5.
  - FST: speaking/listening CLB/NCLC 5; reading/writing CLB/NCLC 4.
- No full score-to-CLB tables here; referenced for program minima only.
