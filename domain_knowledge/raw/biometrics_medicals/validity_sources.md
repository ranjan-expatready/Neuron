# Biometrics & Medicals – Validity/Reuse Sources (DRAFT, NOT LEGAL ADVICE)
> NOTE: Internal engineering reference only. Always cross-check with the live IRCC website.

## SOURCE: Find out if you need to give biometrics (IRCC)
- URL: https://www.cic.gc.ca/english/visit/biometrics.asp
- DATE_MODIFIED (page meta): 2024-03-11
- TYPE: HTML page
- EXTRACTED_TEXT (key points)
  - Biometrics required for most applicants between ages 14–79.
  - If biometrics are still valid from a prior visitor/study/work/PR application, you don’t need to give them again; status can be checked via the “Check Status Tool.”
  - Mentions prior Canadian temporary visa within past 10 years in eligibility questions; implies reuse window tied to prior valid biometrics.
  - Reminds applicants to pay biometrics fee and submit within the request window; failure can delay processing.

## SOURCE: Medical exams for temporary residence (IRCC)
- URL: https://www.cic.gc.ca/english/information/medical/medexams-temp.asp
- DATE_MODIFIED: not shown (accessed 2025-12-08)
- TYPE: HTML page
- EXTRACTED_TEXT (key points)
  - Outlines when temporary resident applicants need an immigration medical exam (IME) and when upfront exams are allowed/required.
  - States: “Your medical exam results are good for 12 months only. If you’re a student or worker, your letter of introduction shows the date your results expire.”
  - Notes that panel physicians must perform exams; validity tied to exam date; new exam required if not using results within validity window.

## SOURCE: Medical exams for permanent residence (IRCC)
- URL: https://www.cic.gc.ca/english/information/medical/medexams-perm.asp
- DATE_MODIFIED: not shown (accessed 2025-12-08)
- TYPE: HTML page
- EXTRACTED_TEXT (key points)
  - For PR applicants: medical exam results valid for 12 months; if landing after that window, a new exam may be required.
  - Confirms panel physician requirement and that results can expire before admission if landing delayed.
  - Highlights that validity applies per exam instance; reuse depends on landing within 12-month window.
