# IRCC Legal Knowledge Base

## Overview

This directory contains comprehensive information about Immigration, Refugees and Citizenship Canada (IRCC) programs, policies, regulations, and procedures. This knowledge base serves as the authoritative source for all IRCC-related information within the Neuron system.

## Directory Structure

```
ircc/
├── _README.md                          # This overview document
├── programs/                           # Immigration programs
├── regulations/                        # Laws and regulations
├── forms/                              # Official forms and guides
├── fees/                               # Fee schedules and information
├── processing_times/                   # Current processing times
└── changelog/                          # Policy and regulation changes
```

### 3. Practical Application
