# Quick Start: Master Prompt

## Copy & Paste Anytime

**Use this prompt for any task:**

```
@Product Manager/CTO Agent:

[YOUR TASK HERE]

Read knowledge base first. Check requirements coverage - what's covered? What's missing? During implementation, identify gaps - research IRCC if legal, ask me if business decision. Update requirements coverage. Use FAANG-style development (ATDD + Contract + TDD). Coordinate with TestSprite Agent and other agents. Update knowledge base. Ensure 80%+ coverage. Report status and coverage progress.

Proceed.
```
